# tracklister

> Find tracks from your favorite music set

See demo: [https://sevic.dev/tracklister](https://sevic.dev/tracklister)

### :wrench: Setup

```bash
git clone https://github.com/zsevic/tracklister
cd tracklister
cp .env.sample .env # change values after copying
yarn
yarn start
```

### :rocket: Deployment

```bash
yarn deploy
```

- React
