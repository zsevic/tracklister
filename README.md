# tracklister

> Find tracks from your favorite music set

See demo: [https://sevic.me/tracklister](https://sevic.me/tracklister)

### :wrench: Setup

```bash
git clone https://github.com/zsevic/tracklister
cd tracklister
yarn
yarn start
```

### :rocket: Deployment

```bash
yarn deploy
```

- React
